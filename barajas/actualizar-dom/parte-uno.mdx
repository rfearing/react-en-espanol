<p key="p1-3">3. CSS proporciona estilos</p>
<p key="p1-4">4. y por último, JavaScript proporciona interactividad.</p>

<p key="p1-5">
  5. Para hacer cambios en las páginas web, Javascript necesita un poco de ayuda. <br />
  Afortunadamente, todos los navegadores modernos proporcionan a Javascript esta ayuda necesaria via Web APIs.
</p>

<p key="p1-6">
  Las Web APIs son un conjunto de herramientas que permiten a Javascript interactuar con el navegador y el usuario.
</p>

<p key="p1-7">
  El API para manipular el DOM HTML, se llama "HTML DOM API".
</p>

<p key="p1-8">
  Alugnos otros Web APIs incluyen el Fetch API, para hacer conexiones de red y el URL API para estructurar y analizar URLs.
